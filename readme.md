# VACG Ambiental

Sistema simples para controle ambiental, desenvolvido para auxiliar no gerenciamento de informações da empresa.

## 📋 Funcionalidades

- Cadastro de atividades ambientais
- Controle de datas e prazos
- Registro de informações importantes
- Armazenamento de dados no navegador usando LocalStorage

## 💻 Tecnologias utilizadas

- HTML
- CSS
- JavaScript


## 🚀 Como acessar

O sistema pode ser acessado pelo GitHub Pages:

https://seuusuario.github.io/VACG-Ambiental/

## 📂 Como usar

1. Acesse o sistema pelo link
2. Cadastre as informações necessárias
3. Os dados ficam salvos no navegador automaticamente
